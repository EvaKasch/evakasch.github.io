郑码输入法 你值得拥有

发表于 2016-03-03 | | 阅读次数

偶遇过一些打字大神，觉得好专业，好羡慕~~ 然而自己的输入速度却难以提升。难道真的是我的手脑协调性过低? Nooooooo!

拼音选字又慢又累

作为一名重度拼音输入法用户，必然忍受拼音输入法高重码率的缺点。比如，输入偏难怪词，常常翻好几页才能找到。这样打字速度就会因选字而大大降低了。另外，如今拼音输入法大多提供智能排序功能，候选词的位置不固定，输入时就不能因对输入法熟悉而背诵词语列表(背诵词语列表是智能ABC大神的必备技)。这样就要盯着候选词框，选择正确的词语。选字真是又慢又累。什么? 你问：“难道还有更轻松的输入法?” 当然啊！

五笔不用选字

为了避免选字的耗时耗心，相当多打字大神选择了大名鼎鼎的五笔。五笔是形码的一种，形码从汉字的结构入手，将汉字分解成若干部件。

形码的优势就是重码率低，当你输入一个字时，往往不需要选择候选项，仅按空格即可输入，这无疑减少了反应时间，加快输入速度，甚至可以解放双眼，实现真正的盲打。

那我最后怎么选择了郑码，没选择五笔呢?

五笔拆字难，难学易忘

事实上，五笔除拼音外最热门的简体汉语输入法，作为盲目从众的我，当然率先试过啦～

然而学习两三天后，不能忍的点渐渐出现——其一是不合理的拆字规则，这显得很不规范，同时不熟的字就很难打出来了。虽然都说98版五笔修正了86版五笔的很多不合理的拆字规则，却又听说“86版五笔更加广泛”，“应从86版五笔开始学习”等等建议，所以有些爱好者提议——最好两个版本的五笔都熟悉。OMG，在逗我?！

难学易忘是五笔又一缺点。五笔的码表是因舒适度排列的，没有逻辑理由，所以如果许久不用，可能需要重新学习。

郑码就很赞啊

微软的Windows 8之前的多版系统都自带“郑码”输入法，可见郑码曾经的影响力（虽然现在用户不多）。

它也是一种形码输入法，与五笔不同的是拆字规范，重码率更低，码表有规律，不易忘记。

正巧我不想选字，又不爱死记硬背，毫不犹豫选择了郑码。好在郑码的学习资料很多，也很正规，自学并不困难。

如果你有兴趣打字提速，愿意记忆一定量的词根，都可以在一天至数月之内掌握郑码，并在日常应用中逐渐提速。郑码是一款学会了就会爱上的输入法，欢迎你的加入~~~

本人的入门方式：安装郑码输入法 -> 阅读《郑码实用手册》，并按照内容练习 -> 用它来聊天 -> 学术写作

资料详见文尾的“资源下载”。

郑码适合自己吗

笔峰一转，客观公正的来说（毕竟没有拿好处费），没有一种输入法是完美的。你更在意的是什么呢?

由于形码需要拆解字，如果不会写，或者不会拆，那么都很难打出。郑码极大的解决的拆字难题，但不会写咋整= =

tip: 有的输入法可以同时输入郑码和拼音，这一定程度可以取长补短。

对拼音掌握不好，又不会提笔忘字的人，最适合学习形码了。什么? 你提笔忘字? 如果你想不再提笔忘字了，欢迎加入郑码:-)

经过了一定量的调研（严肃脸），我为大家奉上整理好的各输入法的特(gan)点(huo)：

拼音的优点

1.易学难忘
2.软件智能
3.不需要会写，会念即可

拼音的缺点

1.选字耗时长，打字速度受限
2.容易泄漏隐私
3.不会念的字打不出

形码的优点

1.击键流畅，输入专心，提升手指和大脑的灵活性，提升打字速度
2.不会念的字也能打出
3.不易泄漏隐私
4.爱上打字

形码的缺点

1.记忆量大，上手慢
2.不会写的字很难打出来

五笔的优点

1.字根的分布考虑按键舒适度
2.字根较少，易记
3.重码率低
4.用户群大，可选输入法软件多，高手多。

五笔的缺点

1.长久不用易忘
2.拆字难
3.字库小

郑码的优点

1.字根的分布有逻辑，易于记忆。
2.拆字易
3.字库大
4.windows 7系统及以前的版本的系统自带输入法

郑码的缺点

1.字根分布没有考虑指法，舒适性有待提高
2.用户少，软件人性化差
3.字根多，记忆量大。
